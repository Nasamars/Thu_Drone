### Ubuntu介绍
Ubuntu (又称乌班图)是一个以桌面应用为主的开源GNU/Linux操作系统，支持
x86、x64、ARM和ppc架构，由全球化的专业开发团队(Canonical Ltd)打造的。
Ubuntu每六个月便会发布一个新版，以便人们实时地获取和使用新软件。推荐使 用长期支持版本(Long Term Support，LTS):16.04、18.04、20.04。
